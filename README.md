## Description

An API powered in NestJS to store and manage notes with tags.

## Project setup

1.
```bash
$ npm install
```

2. Create an .env file with the content of the .env.example file.

3.
```bash
$ npx prisma generate
```
4.
```bash
$ npx prisma migrate dev
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
